How To Upcycle

Before throwing something out, consider how it can be repurposed. Start small, perhaps by turning empty tins into plant pots or personalizing an old piece of clothing.

Use things you already have, go to thrift shops or antique stores. Shop second-hand; check online sales apps such as Facebook Marketplace, Depop, ThreadUp, Vinted, Poshmark, etc.

There are very few limits to what you can create; some things that are typically upcycled are clothes, furniture, and random miscellaneous items. Depending on what you have, you can create decor, clothing and accessories, storage and organizing units and way more!

A Brief Step-by-Step

1.  Find an item you want to upcycle & think of different things to
        turn it into.
2.  Create a plan; use tutorials, find pattern pieces, make sure you
          have all the equipment and materials you need to create your
          project. Remember: measure twice, cut once!
3.   Clean and prepare your item; clean it, iron it, cut off loose/ spare
           parts, etc.
4a.   If you’re painting an item, start by sanding it to create a rough
               surface for the paint to stick to.
4b.   If you’re sewing something, find or make pattern pieces to create
               a cutting template. Cut out your pieces and start sewing (make
               sure you have enough fabric before starting).
5.   Start cutting, sanding, painting, sewing, gluing, and combining all
           of your elements together to create something new.
6.   Seal, hem, and finish your upcycled project.
